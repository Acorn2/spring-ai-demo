# 公司规章制度

## 报销标准

### 餐饮报销
- 餐饮不超过 100 元 / 人 / 天
- 需要提供发票
- 超过标准需要部门经理审批

### 交通报销
- 市内交通：实报实销
- 出差交通：按实际发生费用报销
- 需要提供行程单或发票

## 考勤制度

### 工作时间
- 周一至周五：9:00 - 18:00
- 午休时间：12:00 - 13:00

### 请假流程
1. 提前在 OA 系统提交请假申请
2. 直属上级审批
3. 超过 3 天需要部门经理审批

## 福利待遇

### 年假
- 入职满 1 年：5 天年假
- 入职满 3 年：10 天年假
- 入职满 5 年：15 天年假

### 五险一金
- 按国家规定缴纳
- 基数按实际工资计算
